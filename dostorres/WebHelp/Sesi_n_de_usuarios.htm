Sesión de usuarios:

Al pulsar click en esta opción el usuario al cual se le de la permisología para acceder a este modulo puede observar que usuarios tienen la sesión abierta. La función de esta opción dentro del modulo de administración es la de cerrar la sesión de un usuario correctamente. Cuando un usuario cierra la pantalla del sistema sin salir antes pulsando LOGOUT    el usuario administrador deberá acceder a esta opción pulsar el usuario como se muestra en la pantalla y cerrar la sesión correctamente.
