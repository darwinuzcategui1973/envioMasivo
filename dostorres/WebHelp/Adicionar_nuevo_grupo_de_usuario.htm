Adicionar nuevo grupo de usuario:

Al pulsar click en esta opción el sistema muestra los campos con los campos que deberán ser llenados para crear o adicionar un nuevo grupo de usuario.

Para la Adición de un nuevo grupo de usuario es necesario:

1.      Rellenar las casillas correspondientes a: Nombre del Grupo y Descripción.

2.      Seleccione y Pulse Guardar

 

Nota: Para poder establecer la seguridad a un grupo creado, se debe acceder al modulo Ver / Editar seguridad.
