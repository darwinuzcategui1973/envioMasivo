Bloquear

Esta es una las opciones mas importantes dentro de QWEBDOCUMENTS® y debe ser habilitada solo para aquellos usuarios con el perfil adecuado para tomar el control de cambios de un documento dentro de una localidad o carpeta. Bloquear permite la liberación de un documento que será objeto de revisión y cambios por parte de un grupo de usuarios. El documento al que se le realiza la acción de Bloquear queda bloqueado hasta tanto no se culmine su revisión y su status puede ser cambiado al utilizar las opciones de Modificar o Deshacer Bloqueo.
