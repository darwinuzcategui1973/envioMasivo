Notificaciones emitidas por el sistema:

 

Estas notificaciones son emitidas por el sistema cuando: (Ver Notificar por correo y Valores establecidos para un flujo de trabajo)

1.      Un documento ha expirado, de acuerdo a la vigencia establecida por un usuario o administrador del sistema. (Ver vigencia de Documentos)

2.      Un documento en borrador ha expirado, de acuerdo a la vigencia establecida por un usuario o administrador del sistema. (Ver vigencia de Borradores)

3.      Cuando un flujo de trabajo de aprobación o revisión ha expirado, de acuerdo a la vigencia establecida por el  usuario que lo activa o el administrador del sistema.

4.      Notificación de flujos de trabajo de revisión de un documento, cuando es involucrado un usuario en dicho proceso.

5.      Notificación de flujos de trabajo de aprobación de un documento, cuando es involucrado un usuario en dicho proceso.

6.      Notificación de la cancelación de un flujo de trabajo.

Estos mensajes pueden contener un mensaje personalizado el cual es asignado por el administrador del sistema en los siguientes campos:

1.      Mensaje para documentos ya expirados

2.      Mensaje para documentos borradores ya expirados

3.      Mensajes para flujos de trabajo ya expirados

4.      Mensajes para flujos de trabajo revisión 

5.      Mensajes para flujos de  trabajo aprobados

6.      Mensajes para flujos de trabajo cancelados
