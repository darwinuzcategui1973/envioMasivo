Menú de Localidad/Carpeta/Proceso

Localizado sobre la lista de carpetas/procesos/documentos y al lado derecho de la pantalla de QWEBDOCUMENTS®, las opciones de menú de Localidad/Carpetas/Procesos se cambian dinámicamente, basado en el nivel de estructura y el contexto de permisos de seguridad del usuario.

 

Agregar Localidad

Permite agregar dentro de la estructura de documentos una nueva localidad.

Agregar Carpeta

Permite agregar dentro de la estructura una nueva carpeta de Documentos.

Agregar Proceso

Permite agregar dentro de la estructura una nueva carpeta de Documentos.

Eliminar

Permite eliminar una localidad, carpeta o proceso de la estructura de documentos.

Nota: Solo puede eliminar el nivel deseado, siempre y cuando este vació, es decir, sin documentos cargados y se posea la permisología para hacerlo.

Propiedades

Permite cambiar las propiedades de los documentos, solo los campos que se encuentran disponibles.

Mover

Moviliza una carpeta/proceso, junto con la documentación contenida dentro de este a otra carpeta/proceso.

Nota: Al mover la carpeta/proceso a otro nivel, el código de la documentación actual le cambia, el/los documento(s) trasladado(s) a una carpeta nueva, toma el prefijo de la carpeta nueva y valida; si el correlativo existe lo incrementa en uno mas.

Permisos a Usuarios

Permite visualizar los permisos agregados o dados a través de una ventana de configuración, cuando se dan opciones de seguridad a un usuario este pasa a ser miembro de la lista de usuarios con privilegios sobre la carpeta/proceso/localidad actual o sobre la que fue seleccionada(o).

Permisos a Grupos

Permite visualizar los permisos agregados o dados a través de una ventana de configuración, cuando se dan opciones de seguridad a un grupo de usuarios este pasa a ser miembro de la lista de grupos con privilegios sobre la carpeta/proceso/localidad actual o sobre la que fue seleccionada(o).

Correos sin leer

Permite visualizar el número de correos que un usuario tiene pendientes por leer, ya sea que fueron generados por un Flujo de Trabajo o simplemente le fue enviado por la opción de mensajes.

Bajar última versión

Permite al usuario acceder a la última versión que posee el documento, independientemente del status que posea.

Enviar como Link

Permite que el usuario envíe por mail a otro usuario un link que al hacer click automáticamente le muestre el documento que se desea visualizar, variar propiedades, etc.

Flujo(s) Pendiente(s)

Permite visualizar el número de Flujos de Trabajo (Revisión ó Aprobación) pendientes por ejecutar.
