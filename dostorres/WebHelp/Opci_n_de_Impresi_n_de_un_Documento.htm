Opción de Impresión de un Documento

Dentro de las funcionalidades de QWEBDOCUMENTS® esta la del control de impresión de documentos. QWEBDOCUMENTS® restringe la impresión de documentos solo a los usuarios propietarios o autorizados por el propietario para imprimir documentos. QWEBDOCUMENTS® permite imprimir solo documentos en estatus de Aprobado o Publicado para evitar circular información no actualizada dentro de la compañía.

QWEBDOCUMENTS® permite la impresión de copias controladas y no controladas. En la opción de Copias controladas el usuario solicitante puede definir los usuarios destinos y la cantidad de copias solicitada para cada uno de ellos. Para la opción de Copias No Controladas el usuario solo podrá definir los usuarios destinatarios.
