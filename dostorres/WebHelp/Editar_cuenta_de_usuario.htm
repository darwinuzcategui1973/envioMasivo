Editar cuenta de usuario:

Pulsar click en Editar cuenta usuario permite visualizar una lista de todos los usuarios existentes en QWEBDOCUMENTS® mostrando el ID de Usuario, El nombre del Usuario y el Grupo al cual pertenece.

Para modificar los datos de un usuario solo se debe hacer click sobre el ID de usuario y esta desplegara la pantalla con el registro del usuario y todos sus datos, como se muestran en las pantallas a continuación.  Es necesario destacar que si el usuario va a ser eliminado deberá ser escogido el nuevo usuario que va a ser propietario de sus documentos.
