Requerimientos de Hardware y Software

 

Para poder optar por el programa de QWEBDOCUMENTS® se recomienda como mínimo, tanto para servidores como para estaciones de trabajo los siguientes requerimientos:

 

SERVIDOR

Servidor – Hardware

Procesador

Pentium lV 1.0 + GHz

Memoria Ram

1+ GB. La cantidad de memoria

Ram varia según la cantidad de

usuarios Web, así como de la

cantidad de tareas extras que

ejecute el servidor.

Disco Duro

500MB de espacio libres para

instalación del producto y

configuración básica.

6 GB de espacio libre para Bases

de Datos

 

Servidor - Software

Sistemas Operativos:

• Windows 2000 Professional,

Server o Advanced Server

con Service Pack 3 o

superior.

• Microsoft SQL Server 2000

con Service Pack 3 o

superior.

 

 

 

 

ESTACIÓN DE TRABAJO

Estación – Hardware

Procesador

Pentium III 500+ MHz.

Memoria Ram

512 MB.

Disco Duro

500 MB de espacio libres.

 

Estación - Software

Sistemas Operativos

Microsoft Windows XP, 2000

Profesional.

Browser (navegador)

Netscape ó Microsoft Internet

Explorer V. 5.5 o superior.

Paquete de Aplicaciones

MS Office 97, 2000, XP (Word,

Excel, PowerPoint).

Visualizadores de otras

aplicaciones necesarias.

(Microsoft Project, Visio,

Autocad, Etc.)

 

Algunas de las razones que pueden hacer variar estos valores son:

 

• Cantidad de usuarios

• Otras aplicaciones que corran en el servidor

• Tamaño y cantidad de los documentos

• Rol del usuario que utilice la estación de trabajo

• Web App. Server y el RDBMS compartiendo el servidor

 

La resolución recomendada para la utilización de QWEBDOCUMENTS® es 800x600 píxeles. Una resolución menor puede provocar una visualización incorrecta, o la perdida de opciones necesarias.
