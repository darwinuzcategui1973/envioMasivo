Adicionar nueva cuenta de usuario:

 

 

Al pulsar click en esta opción el sistema muestra una ficha con unos campos que deberán ser llenados con la información del usuario.

 

Para la Adición de nuevas cuentas de usuarios es necesario:

 

1.    Rellenar las casillas correspondientes a Nombre, Apellido, Cargo, Contraseña, e-mail.

     Nota: Para que esta  opción puede ser realizada es requisito que el usuario tenga una cuenta de correo electrónico.

2.    Seleccione el grupo de usuario y área al cual pertenecerá la nueva cuenta (Ver grupos de usuarios).
Nota: Al seleccionar el grupo, el usuario heredara automáticamente la permisología que le fue asignado a dicho grupo en el sistema (Ver / Editar seguridad).

3.    Seleccione y Pulse Guardar.

 

Nota: Estos datos deben llenase cuidadosamente ya que las mismas servirán para la correcta identificación del usuario en el sistema
