Para visualizar una lista de todos los usuarios o Grupos que tienen asignada seguridad para el elemento seleccionado:

La primera lista de Usuarios/Grupos desplegada es la lista de todos los usuarios existentes en QwebDocuments y la segunda lista identifica los usuarios con seguridad en el elemento seleccionado. Permite que el usuario verifique rápidamente que usuarios poseen configuración de seguridad y cuales no.

1.      Si desea verificar los permisos de seguridad, seleccione un usuario/Grupo. La configuración de seguridad del usuario/grupo seleccionado es desplegada (Vista efectiva de permisología).

2.      Una vez disponible la Vista podrá modificar la configuración de seguridad para el elemento actual.
